# User Story: Sección Portfolio de Proyectos n8n

## Título
Como visitante de pedrocastro.pro, quiero ver una sección dedicada a proyectos y automatizaciones n8n para entender las capacidades técnicas de Pedro y los resultados que puede entregar.

## Descripción
Crear una sección `n8n.pedrocastro.pro` que funcione como portfolio técnico especializado en automatizaciones con n8n, mostrando proyectos reales, casos de uso, y demostrando expertise en AI automation.

## Contexto de Negocio
- **Objetivo**: Convertir visitantes en leads cualificados que entiendan el valor de las automatizaciones con IA
- **Audiencia**: Fundadores de startups, CTOs, responsables de operaciones buscando optimizar procesos
- **Diferenciador**: Mostrar casos reales con resultados medibles, no solo teoría

## Criterios de Aceptación

### Must Have (P0)
- [ ] Hero section con propuesta de valor clara
- [ ] Mínimo 3 proyectos showcase con:
  - [ ] Título y descripción del problema resuelto
  - [ ] Screenshot o diagrama del workflow
  - [ ] Tecnologías utilizadas (n8n, LangChain, APIs específicas)
  - [ ] Resultados medibles (tiempo ahorrado, reducción de errores, etc.)
- [ ] Sección de "Casos de Uso Comunes":
  - [ ] Automatización de soporte cliente
  - [ ] RAG systems para documentación
  - [ ] Integración Gmail + IA
  - [ ] Generación automatizada de contenido
- [ ] CTA claro para agendar auditoría gratuita de 30 min
- [ ] Responsive design (mobile-first)
- [ ] Tiempo de carga < 2 segundos
